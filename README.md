# Projection Prediction Model

A machine learning-based application that forecasts key performance metrics (clicks, leads, conversions) based on marketing budget inputs. Built to help marketers and analysts simulate scenarios and make data-driven decisions.

---

## Overview

This project focuses on predicting campaign performance by enabling users to input different budget values and instantly visualize expected outcomes, making it easier to optimize marketing strategies.

---

##  Problem Statement

Marketing teams often struggle to estimate the impact of budget allocation on campaign performance. This project solves that by building a predictive model that learns patterns from past data and generates reliable forecasts.

---

##  Tech Stack

* **Language:** Python
* **Machine Learning:** Scikit-learn (Linear Regression, Ridge Regression)
* **Data Processing:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Deployment:** Streamlit

---

##  Features

*  Predicts clicks, leads, and conversions from budget input
*  Scenario simulation for multiple budget strategies
*  Interactive dashboard using Streamlit
*  Visual insights on ad spend vs performance
*  Regression-based forecasting models

---

## Model Details

* Implemented **Linear Regression** and **Ridge Regression** models
* Performed feature engineering to improve model performance
* Evaluated models using standard regression metrics (R² Score, MAE)
